# Sistema de Gestión de Reservas - Software FJ

## Descripción

Este proyecto fue desarrollado como parte de la **Fase 4** del curso **Programación** de la Universidad Nacional Abierta y a Distancia (UNAD).

La aplicación permite gestionar clientes y reservas para los diferentes servicios ofrecidos por la empresa **Software FJ**, implementando los principios de la **Programación Orientada a Objetos (POO)** en Python. Además, incorpora una interfaz gráfica desarrollada con **Tkinter**, manejo de excepciones personalizadas, registro de eventos mediante *logging* y control de versiones con Git y GitHub.

---

## Funcionalidades

- Registro de clientes.
- Gestión de reservas.
- Reserva de salas.
- Alquiler de equipos.
- Asesorías especializadas.
- Interfaz gráfica desarrollada con Tkinter.
- Validación de datos mediante excepciones personalizadas.
- Manejo de errores utilizando `try`, `except`, `else` y `finally`.
- Registro de eventos utilizando `logging`.
- Simulación de diferentes escenarios de uso.
- Código modular y orientado a objetos.

## Cómo ejecutar el proyecto

1. Clonar el repositorio.

2. Abrir la carpeta del proyecto.

3. Ejecutar el siguiente comando:

```bash
python main.py
```

4. Desde la interfaz gráfica es posible:

- Registrar una reserva.
- Seleccionar el tipo de servicio.
- Ejecutar las simulaciones del sistema.
- Visualizar mensajes de confirmación y error.
