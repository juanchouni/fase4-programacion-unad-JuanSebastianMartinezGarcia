# Sistema de Gestión de Reservas - Software FJ

## Descripción

Este proyecto fue desarrollado como parte de la Fase 4 del curso de Programación de la UNAD.

El sistema permite gestionar clientes y reservas para los servicios ofrecidos por la empresa Software FJ, aplicando los principios de la Programación Orientada a Objetos en Python.

## Funcionalidades

- Registro de clientes.
- Gestión de reservas.
- Reserva de salas.
- Alquiler de equipos.
- Asesorías especializadas.
- Validación de datos mediante excepciones personalizadas.
- Registro de eventos utilizando logging.
- Simulación de diferentes escenarios de uso.

## Tecnologías utilizadas

- Python 3
- Git
- GitHub
- Visual Studio Code

1. Clonar el repositorio.

2. Abrir la carpeta del proyecto.

3. Ejecutar:

```bash

```
